# MindFusion.Diagramming for JavaScript - NPM Starter Project

## About The Project

A starter project that shows how to use the MindFusion.Diagramming for JavaScript library with NPM and Vite.

### Installation

Clone the repo
   ```sh
   git clone https://github.com/MindFusionComponents/diagram-starter-npm
   ```
Install NPM packages
   ```sh
   npm install
   ```

## Getting Started

Run the following command to start the development server:
   ```sh
   npm run dev
   ```
